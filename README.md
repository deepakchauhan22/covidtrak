# Covidtrak - A Live Covid-19 Corona stats Tracker

## www.covidtrak.in App Fetches Location wise covid-19 Stats
### This Project is Using Stats API from covid19india.org

 **Tech Stack**
   ```bash
* JavaScript
* React
* Geolocation API 
* Amazon S3
* Chart js
* html
* css
```
